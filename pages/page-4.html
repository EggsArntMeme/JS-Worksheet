If Statements

conditional statements! if you can handle it... hehe

  • What is an If Statement

    An if statement is a conditional statement, basically meaning, if condition is met (or is true) then do whatevers inside! If statements take a lot from just simple english sentance building, so its easier to understand. If the statement is getting too confusing, maybe its best to split it up or it might just be a complex question :/

  • How would you write it?

    as stated before, if statments are really simple to write, just needing an if at the beginning, your condition in round brackets and then squiggly brackets with the code we want to run if the condition is true! Now. There are key works and symbols you can use to change up how the if statement will work (although its all still 'if true')

  • and, or, not!

    these words, 'and', 'or' and 'not' are the special words, first off is going to be not, as its the simplest. Not reverses the if statement, simply checking if the statement is NOT true, basically doing the reverse!

  • Conditional Statements

    obviously there are multiple ways to actually write a check if something is true or not, so 9 times out of 10 will use symbols to compaire the value of a variable, 1 other time is when we know the value or return is a bool, if it is, we can just keep it by itself (we dont need to check if the bool (which is only ever true or false) is true or not)

  • Comparison Symbols

    there are 6 total symbols, (but honestly half of them are the same) AND 4 INVOLVE MATH :D.... anyways.

  • "=="

    First one is '==' (two equal signs) this checks if the two components are the exact same (although if it checks for exactly the same changes from language to language). [don't confuse this for a singular '=', two is to compare, one is to set!]

  • "!="

    Just like the last one, this checks if the two components are the same, however, this time it runs through if its false and doesn't if its true! Just like the 'not' from before (however if we stack the two it would just be the same as "==")

  • ">" and "<"

    While it looks like two birds looking at the tasty and, this is the two next signs, that checks if the the value thats being compared is greater than or less than, just think of it like how math uses them! (did I lump them together cause I couldn't remember what name goes to which one? ....I wont tell.)

  • "=>" and "=<"

    I dont even have a joke for a face for the second one. BUT THESE ARE THE LAST TWO!! these just compare if the value is greater than or less than and equal to the value, honestly isn't that special in anyway lol

  • Else Statements

    Else is added on just at the end of an if statement just outside of the squiggly bracket to go over "else / otherwise do this" you can also add on another if infront of the else to make something called an 'elif' statement, where "else if ___" to which you can also add on an else statement on the end of that too. Else doesn't take a condition, and its just a catch all for if statements.

if (condition) { . . . }

and

if (condition) {. . .} else { . . . }
