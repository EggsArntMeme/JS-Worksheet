  • What are Objects?

    Objects are a form of holding information through properties, these properties hold any type of information (string, int, float, etc.) and assigns it a key or name that we can use to get that specific information.

  • How do I get the information out?

    just like with lists, we can access the information inside by adding square brackets at the end. However, this time we cant use an index number to collect it. Instead, we use a key, the key is the name that we set up before hand (you can also use objectVar.name to select the info)

  • Can I place Objects in lists?

    Yes! and fun fact, if you format it in a specific way, thats called a JSON file (Java Script Object Notation). These are used to hold and collect information. Nine times out of ten you won't write JSON files, you'd just create them and 'write' them through code, or they'll just be supplied to you.

objectVar = { name: value , . . .}

and

objectVar.name or objectVar[name]
