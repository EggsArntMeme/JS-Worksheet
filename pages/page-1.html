Variables

As the first thing we're gunna review is Variables.

  • What is a Variable?

    Variables are basically containers that hold information, they are also the keystone to every language. They can hold many things and thats what make them useful!

  • How do we use them?

    To use them, we set them to a reasonable name using "const" or "let" we use const when the value won't be changed, and will be const-ant. Meanwhile, let is used when the variables value will change, thusly we're let-ing its be this value for now.

let variableName = value

or

const variableName = value
