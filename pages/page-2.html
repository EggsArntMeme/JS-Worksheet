Functions

next, we are going over functions.

  • What is a Function?

    A function is basically a set of instructions we set up that can be called on and used later! Normally, with other languages, you have to define or describe the function before you use it (thusly its common to see them at beguinning of scripts) However, JS doesn't care about that, so we can define our function whereever, as long as its not already within one, otherwise it becomes a local function

  • What is a "Local Function" and what does it mean?

    the defining word "local" covers anything defined within a function (as well as other things, but we haven't gotten that far yet ;P) So if you were to set a variable within a function, that would be concidered a "local Variable" and can not be accessed in other functions or anywhere but in its origin function.

  • So how do we get that value out?

    for that we use the 'return' statement! a return statement will return whatever value was held within the variable or condition. using this we can get an output of a function and use it in calculations, functions, or just to simplify other code!

  • How do I get a variable in a function?

    for that, we use whats called Parameters (or called param for short), simply put, its a variable that we're assuming the input of (let's say an integer) then we can do calculations using the assumed variables and return a sum of two numbers

  • Well how do you actually write a Function

    There are 3 main ways to writing a function... However, I may or may not have forgotten what one of the ways is, so thus I'll just show off the two ways I do know. [Note: see how the first way looks just like we're setting a variable using const, like we're setting a variable and the value is actually just instructions!]

const functionName(param) => { . . . }

or

function functionName(param) { . . . }
