Next, we'll learn about loops! Next, we'll learn about loops! Next, we'll learn about loops! Next, we'll-

  • What are Arrays (or lists)?

    Quick preface that I will likely call Arrays as lists, just what im more used too. ANYWAYS. Lists are a type of variable that holds multiple values, sorted by an index! to create a list, you simply save a variable with apair of square brackets ( '[]' <- these) These values are seperated by a comma, just like your listing objects.

  • What can Arrays hold?

    As said before, lists can hold literally all values so it can hold: integers, floats, strings, booleans, variables, and even other arrays! having a list that contains multiple lists is often refered to as a '2D list' however that is more complex than we'll cover for now.

  • How do I select stuff from lists?

    to pick out specific values from lists, we often use the index (ooooo). The lists index starts at 0, then increase by 1! To select specific values, you add the index within square brackets simply like: listVariable[0] (:O gasp)

  • What are For loops?

    For loops are a type of loop that goes through the code within it until specified condition is met (for loops are also iterative in nature, going through values one by one through a list).
