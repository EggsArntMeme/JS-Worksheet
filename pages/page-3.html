Time for the part most of us got lost haha, Events!

  • What is an Event?

    Events are whats happening on your page, or what the user is currently doing. We can listen to events by using an eventListener, which will look out for whenever the action is done, and use an action

  • How would you write it?

    Since Event listeners are similar to functions and take parameters! Specifically, it takes two, the action done, taken in a string, and a function. All actions that can be listened for are pre determined, so simply looking at the documentation for eventListeners can show you all the specific actions we can look out for! [NOTE: you can check it out Here :3

  • Hold on. This sounds familiar

    If you thought this, you'd be correct! Event listeners are like a function in the sense that as soon as it's called, it runs specific code (pre determined by us), and actually to specify what code will run, we do use functions! Almost like an if statement listening for if (let's say) a click happens, then, run this function!

[element].addEventListener('event', func)

or

[element].addEventListener('event', () => { })
